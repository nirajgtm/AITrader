# Case Builder (one-sided, briefed blind)

## When to invoke
A thesis is genuinely contested and I want a clean, hard one-sided case to weigh against the
other side. Usually invoked twice: one for, one against, each blind to the other.

## Inputs / brief (BLIND)
The question and the facts only, plus which side to argue. NEVER my lean, never the other
side's case. The point is a real, undiluted argument.

## Output
The strongest honest case for the assigned side: the core argument, the evidence, what must be
true for it to hold, and the single best point. The mind synthesizes both sides and decides.

## Model
default.
